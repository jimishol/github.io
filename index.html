A Dark Ananke theme

All necessary files are copied from ananke/layout folder to corresponding project’s/layout folder and modified there. So, in order to have a dark theme, copy the below files. (Delete extention .txt, if you directly copy them.) You do not have to modify them. project’s/layout/index.html. project’s/layout/_default/list.html. project’s/layout/_default/summary-with-image.html. project’s/layout/_default/summary.html. project’s/layout/_default/taxonomy.html. project’s/layout/post/list.html. project’s/layout/post/summary.html. Last, make sure your config.toml has the following color parameters you like [params] text_color = "white" background_color_class = "bg-black" body_classes = "avenir bg-black" aux_text_color = "silver" bg_button_color = "bg-black" bg_button_hover_color = "hover-bg-dark-gray" I picked color names from here.
